# Netflix--EDA-Project
Netflix data analysis using Python, Pandas, and visualization libraries
Netflix Data Analysis Project 🎬

📌 Overview

This project is an Exploratory Data Analysis (EDA) of the Netflix dataset using Python. The goal is to analyze content trends, categories, and patterns in Netflix movies and TV shows.

🛠 Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

📊 Project Features

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of trends and patterns
- Insights extraction from Netflix dataset
